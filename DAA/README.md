# DAA — Design and Analysis of Algorithms Lab

Lab programs for DAA subject, written in **Java**.

## 📁 Structure

```
src/
├── LAB1_25DEC29/    # Lab 1 - Dec 29, 2025
├── LAB2_26JAN05/    # Lab 2 - Jan 5, 2026
├── LAB3_26JAN12/    # Lab 3 - Jan 12, 2026
├── LAB4_26JAN19/    # Lab 4 - Jan 19, 2026
├── week1/ – week6/  # Weekly practice programs
└── practise/        # Additional practice
```

## 🚀 Running

```bash
# From the DAA directory
javac src/<folder>/<File>.java -d bin
java -cp bin <ClassName>
```

## 📝 Topics Covered
- Merge Sort
- DFS (Depth First Search)
- Object-Oriented Java concepts (Constructors, Classes)
