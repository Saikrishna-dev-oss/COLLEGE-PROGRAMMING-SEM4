# 📚 SEM 4 — Lab Programs

Repository containing lab programs and assignments for Semester 4 subjects.

---

## 📂 Subjects

### 1. [DAA — Design and Analysis of Algorithms](./DAA/)

- **Language:** Java
- Weekly lab programs covering algorithm design techniques
- Lab assignments organized by date (`LAB1`, `LAB2`, ...)
- Topics include: Merge Sort, DFS, and more

### 2. [OS Lab — Operating Systems](./os_lab/)

- **Language:** C
- Weekly programs for OS concepts
- Topics include:
  - FCFS, SJF, Priority, Round Robin scheduling algorithms (Week 1)
  - More OS topics across weeks 2–6

### 3. [WT Lab — Web Technologies Lab](./wt_lab/)

- **Language:** HTML, CSS, JavaScript
- Lab programs (`p1` – `p10`) covering web development fundamentals
- Assignments and class exercises included



---

## 🛠️ How to Run

### Java (DAA)

```bash
cd DAA
javac src/<LabFolder>/<FileName>.java -d bin
java -cp bin <ClassName>
```

### C (OS Lab)

```bash
cd os_lab/week<N>
gcc <filename>.c -o output
./output
```

### Web (WT Lab)

Open the `.html` files directly in a browser, or use a local server:

```bash
cd wt_lab/lab_programs
# Open p1.html in browser
```
